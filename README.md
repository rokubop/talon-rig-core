# Rig Core

![Version](https://img.shields.io/badge/version-0.6.4-blue)
![Status](https://img.shields.io/badge/status-experimental-orange)
![License](https://img.shields.io/badge/license-MIT-green)

Shared core library for [talon-mouse-rig](https://github.com/rokubop/talon-mouse-rig) and [talon-gamepad-rig](https://github.com/rokubop/talon-gamepad-rig). Provides base classes, lifecycle, layer groups, mode operations, and animation infrastructure.

This repo does nothing by itself - it's meant to be used as a dependency by other rigs.

## Installation

Clone this repo into your [Talon](https://talonvoice.com/) user directory:

```sh
# Mac/Linux
cd ~/.talon/user

# Windows
cd ~/AppData/Roaming/talon/user

git clone https://github.com/rokubop/talon-rig-core
```
